JAVASCRIPT

JavaScript (JS) is a high-level, interpreted programm ing language that is one of the core technologies of the web, alongside HTML and CSS. Originally developed to make web page interactive, it has grown into a powerful, versatile language used both on the client-side and server side.



USES OF JAVASCRIPT

1. Interactive Web Pages :
Used to create dynamic content (e.g image sliders, pop-ups, animations).
Also used to handle user input and events (e.g form validation, button-clicks).

2. Web Development (Front-end) :
Used with framework like React, Angular and Vue. js to build complex single-page application.

3. Mobile App Development :
Frameworks such as React Native, Ionic, and NativeScript let you write JavaScript (and often HTML/CSS) that compilesdown to native iOS or Android components. This “write once run anywhere” approach speeds development and allows sharing code between web and mobile platforms.

4.Game Development:
used to create interactive Graphics and Animation, handle user inputs, handle game logic and rules, control audio and sound effects, control motion and storing game data.
